# 清除缓存

`shield clean` 命令用于清除本地缓存的凭证信息。

## 使用方法

```bash
shield clean
```

## 清除内容

执行后会删除：

- 本地凭证文件（连接器名称、Token 等）
- 下次连接时会自动重新生成新的凭证

## 何时需要清除

- 更换了服务端账号
- 凭证文件损坏导致连接失败
- 需要重新获取连接器身份
- 排查连接问题时作为重置手段

## 注意事项

- 清除后不影响已保存的应用配置
- 下次连接时会自动生成新的凭证，无需额外操作
