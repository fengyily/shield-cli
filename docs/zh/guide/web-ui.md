# Web UI 模式

Web UI 是 Shield CLI 推荐的使用方式，提供浏览器中的图形化管理面板。

## 启动

```bash
shield start
```

默认在 `http://localhost:8181` 启动。如果端口被占用，可以指定其他端口：

```bash
shield start 9090
```

## 功能概览

### 应用管理

- **添加应用**：配置协议、IP、端口和显示名称
- **编辑应用**：修改已保存的应用配置
- **删除应用**：移除不需要的应用
- **重命名**：快速修改应用显示名称
- **最多 10 个应用**：本地加密存储

### 连接控制

- **一键连接**：点击应用卡片上的连接按钮
- **一键断开**：点击断开即可关闭隧道
- **最多 3 个并发连接**：同时连接多个应用
- **自动打开浏览器**：连接成功后自动在新标签页打开 Access URL

### 状态监控

应用卡片实时显示连接状态：

| 状态 | 说明 |
|---|---|
| 未连接 | 应用已保存但未建立隧道 |
| 连接中 | 正在建立隧道和激活服务 |
| 已连接 | 隧道已建立，可通过 Access URL 访问 |
| 连接失败 | 隧道建立失败，查看错误信息 |

状态每 2 秒自动刷新。

### 主题切换

支持深色和浅色两种主题，点击右上角切换。

## 使用场景

Web UI 模式最适合：

- 日常办公，需要管理多个远程应用
- 非技术人员，偏好图形界面操作
- 需要频繁连接/断开不同服务
- 想要保存常用应用配置，避免重复输入
