# 应用配置

Web UI 模式下，Shield CLI 支持保存和管理多个应用配置。

## 应用档案

每个应用配置包含以下信息：

| 字段 | 说明 |
|---|---|
| 协议 | SSH / RDP / VNC / HTTP / HTTPS / Telnet |
| IP 地址 | 目标服务的内网 IP |
| 端口 | 目标服务端口 |
| 名称 | 自定义显示名称（如「办公室电脑」） |
| 认证信息 | 用户名、密码（加密存储） |

## 限制

- 最多保存 **10 个** 应用档案
- 最多同时连接 **3 个** 应用

## 存储方式

应用配置使用 AES-256-GCM 加密后存储在本地：

| 平台 | 存储路径 |
|---|---|
| macOS / Linux | `~/.shield-cli/` |
| Windows | `%LOCALAPPDATA%\ShieldCLI\` |

配置不会上传到服务器，完全本地管理。

## 管理操作

在 Web UI 中可以：

- **添加** — 点击添加按钮，填写应用信息
- **编辑** — 修改已保存的应用配置
- **删除** — 移除不需要的应用
- **重命名** — 快速修改显示名称

每个应用会记录创建时间、修改时间和最后连接时间。
